EQUINE, the Diablo 1 Mod Manager

EQUINE (Easy & QUick Installation Environment is a mod manager for the classic Diablo game.

It has a lot of great features, like auto mod updating, game force updater, downgrader, built in IPXWrapper, 2 DDrawWrappers and much much more.

System Requirements:
Windows 7 SP1 or higher
Diablo or Hellfire Installed
.net Framework 4.8 (Windows 10 users should have this already)
Diablo 1 and/or Hellfire (GOG version recommended)
